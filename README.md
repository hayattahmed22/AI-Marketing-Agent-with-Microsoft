# 🚗 AI Marketing Agent with Microsoft

### Overview
The **AI Marketing Agent with Microsoft** is a smart AI-powered automation system built for Microsoft.  
It automates social media campaigns, generates captions from car catalogues, and creates AI-driven image and video marketing content for businesses.

---

### 💡 Features
- **AI Caption Generator** – Instantly writes ad captions from car or product catalogues.  
- **AI Image Enhancement** – Polishes raw images to make them social-media ready.  
- **AI Video Automation** – Converts catalogue data and images into short marketing videos.  
- **Social Media Automation** – Automatically prepares and schedules posts for different platforms.  

---

### 🧰 Tech Stack
- **Languages:** Python, JavaScript  
- **Frameworks:** Flask, HTML, CSS  
- **AI Tools:** OpenAI API, Microsoft Azure Cognitive Services  
- **Automation Tools:** n8n, Power Automate  
- **Version Control:** Git & GitHub  

---

### 🧑‍💻 How It Works
• Extracts and analyzes catalogue data to generate personalized, high-converting captions tailored to each vehicle and target audienc
1. Upload a car catalogue or raw product images.  
2. The AI reads and processes the data.  
3. It generates captions, enhanced images, and marketing videos automatically.  
4. The results are ready for download or social media posting.  

---

### 📸 Example Use Case
A car dealership uploads raw auction images.  
➡️ The AI agent:
- Enhances image quality  
- Writes ad captions  
- Creates promotional videos  
- Automates posting for Instagram or TikTok
